# To-do API

A simple CRUD Rest API with a basic HTML page to consume it and unit tests.

## Technologies

- ASP.Net
- MVC
- Web API
- REST
- Swagger
- Entity Framework 
    - In memory database
- Unit tests (xUnit)
- HTML/CSS/Javascript
